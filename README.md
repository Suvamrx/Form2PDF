# Bank Transfer Letter PDF Generator

A small Streamlit app that collects transfer details and generates a PDF letter for a bank.

## Features
- Fill in bank and sender details
- Add one or more beneficiary accounts with amounts
- Generate and download a PDF letter

## Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Notes
- Review the generated letter before using it with your bank.
- You can extend the form fields if your bank requires extra details.
